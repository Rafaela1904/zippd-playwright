# Zippd Playwright Automation

This project contains end-to-end automated tests for the Zippd portal using Playwright.

## 🔧 Tech Stack
- Playwright
- TypeScript
- Page Object Model (POM)

## ✅ Features
- Automated login test
- Geolocation handling
- Custom fixtures (hook.ts)

## ▶️ Run Tests
```bash
npx playwright test
